dxleposervice.service module¶

class dxleposervice.service.EpoService(config_dir)¶

Bases: object

A DXL service that exposes the remote commands of one or more ePO servers to the DXL fabric. When a DXL request message is received, the remote command is invoked on the appropriate ePO server and its response is packaged and returned to the invoking client via a DXL response message.

destroy()¶

Destroys the ePO service. This will cause the ePO DXL service to be unregistered with the fabric and the DXL client to be disconnected.

run()¶

Starts the ePO service. This will load the configuration files associated with the service, connect the DXL client to the fabric, and register the ePO DXL service with the fabric.
